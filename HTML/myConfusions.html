1. Linking pages vs. linking resources  
                the head tag is meant for metadata and resources that the browser needs to set up the page.  
                Examples:  
                    <head> CSS files (<link rel="stylesheet" href="style.css">)  
                    JavaScript files (<script src="script.js" defer></script>)  
                    Metadata (<meta charset="UTF-8">)  
                    Page title (<title>My Page</title>)  
                    Navigation links between pages (like <a href="about.html">) are content for the user.  
                    These are visible on the page.  
                    Anything users click or see goes inside <body>.
