        The <meta> tag provides metadata about the HTML document.
        Metadata is “data about data,” meaning information about your web page 
        that does not appear directly on the page but is used by browsers,
        search engines, and other services.

< b > (Bold)
    Purpose: Just for visual styling—makes text appear bold.
    Semantics: None. It doesn’t indicate importance or any special meaning.
< strong >
    Purpose: Indicates strong importance or emphasis.
    Semantics: Has meaning for assistive technologies (like screen readers, SEO). Screen readers might read <strong> text with emphasis.
< i > (Italic)
    Purpose: Just for visual styling—makes text italic.
    Semantics: None. Doesn’t imply emphasis or meaning.
< em > (Emphasis)

Purpose: Indicates emphasized text—something that should be stressed.
Semantics: Important for screen readers and accessibility. Screen readers might change tone or pause slightly.

The <pre> tag in HTML stands for “preformatted text”. Its main purpose is to display text exactly as you type it, preserving all:
    - Spaces (including multiple spaces or tabs)
    - Line breaks (new lines)
    - Indentation

we can also use anchor tags to redirect towards the next page
we can use # in attribute href 
    for example:- """ <a href="#">comming soon</a> """
    In the above example, # acts as a placeholder — it will not redirect to another page.
    Clicking it will jump to the top of the current page.
    Similarly, using #top works the same way if there is an element with id="top" on the page.

What is href?
    href stands for Hypertext Reference.
    It specifies the URL or link where the anchor should navigate.


What is target?
    target specifies where the linked page should open.
    Common values:
        _blank → opens in a new tab or window
        _self → opens in the same frame/tab (default)
        _parent → opens in the parent frame
        _top → opens in the full body of the window, breaking out of frames

           The default value of the target attribute is: _self
           This means if you don’t specify target, the link will open in the same tab or window where the link was clicked.

The alt attribute in the <img> tag is used to describe the image for screen readers 
(to help visually impaired users),and is also used by search engines 
like Google to understand and index images.
    🧏 Accessibility: Screen readers read out the alt text to describe the image for users who can’t see it.
    🔍 SEO (Search Engine Optimization): Search engines use the alt text to understand what an image represents — this helps the image appear in Google Image Search results.
    ⚠️ Fallback: If the image fails to load, the alt text is displayed in its place.
