Input types in form

  1. text

    excepts both numbers and textual data

  2. password

    excepts both it is same as text but input is hidden and shown in form of dots

  3. email

    will not accept wrong input other than email, takes input in email format

  4. numbers

    only takes number, we can set the input range, also in mobile it opens only numpad

  5. date

    can be used to take date input gives calendar option to input date

  6. color

    it can be used to select an color pallete
    also we can use an attribute value="#ff0000" it can be anything
    this sets the default value

  7. file

    u can select file and upload it for that file input can be done
    we can use multiple as well to take multiple input there

  8. radio

    can be used to select mcq type values like gender so that user can select one option among given ones
    can be used for something like genders
    Note:radio's of same type like gender must share same name="" attribute in input

  9. checkbox

    simillar to radio but allows multiple checked boxes

  10. range

    an slider can be used by range and it has min, max and value (the default value)
    as an example: volume slider or brightness slider can be used for the type : range

Note

  1. "placeholder" <-attribute-> for giving hints in input fields

    Note:its only hint not used for an whole instruction so use wisely

  2. "required" <-attribute-> will make an element field must to fill out before sumbiting

  3. padding can be used for the changes in the gapping inside the box for example if i change the padding for the color field then if will shift the color shown inside the box based on the given padding

  4. margin can be used for the changes in the gapping of the box hence outside the box example example if i change the margin for the color field then if will shift the color box bassed on the given margin
