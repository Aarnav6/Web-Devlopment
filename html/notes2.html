1. we can use the multiple classes in an element
    such as .highlight .customColourForLinks etc these can be applied to elements example:
    <p class="highlight customColourForLinks">
  2. we can use various types of selectors in css such as
    1. element selector: directly typing element name like body{}
    2. class selector( . ): we can use an . symbol to call with class name like .highlight{}
    3. id selector ( # ): we can use an # symbol to call with the id name like #importan_note1
    4. universal selector ( * ): we can use an * symbol to make changes universal and it will affect every single element in the file which is to that css
    5. Group Selector : we can use Group collection in css we can use , to seprate selections and they can have same type of class
      example selecting by id #first, #second{} or by class .class1, .class2{}
    6. Nesed Selector ( > ): it can be used to nest the css elements
      example #first>.class1>p{}
      > : this operator in this context is known as child combinator

      In Mordern Css This is Valid

      #first {
      > .class1 {
      > p {
      /* Your styles go here */
      color: green;
      }
      }
      }

  3. priority in the css is :
    Rule 1: based on thr selector :
    Selector Types Example Specifcity Value
    Inline styles <p style="color:red"> 1000
    ID selector #id 100
    Class / attribute / pseudo-class .class, [type="text"], :hover 10
    Element / pseudo-element p, ::before 1
    Universal selector * 0
    Rule 2: If two rules have equal importance and specificity, the last one written in the CSS file wins.
