1. we can use the multiple classes in an element
    such as .highlight .customColourForLinks etc these can be applied to elements example:
    <p class="highlight customColourForLinks">
  2. we can use various types of selectors in css such as
    1. element selector: directly typing element name like body{}
    2. class selector( . ): we can use an . symbol to call with class name like .highlight{}
    3. id selector ( # ): we can use an # symbol to call with the id name like #importan_note1
    4. universal selector ( * ): we can use an * symbol to make changes universal and it will affect every single element in the file which is to that css
