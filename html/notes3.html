JavaScript Arrays, Sets, Freezing, and .length – Notes

1. Arrays in JavaScript

Arrays are ordered collections of elements and can store different types of data (numbers, strings, objects, etc.). Similar to Python lists.

// Example
const arr = [1, "hello", true, { name: "Aarnav" }];

2. .length in JavaScript

The .length property gives the number of elements in an array or characters in a string.


const arr = [1, 2, 3];
console.log(arr.length); // 3

const str = "hello";
console.log(str.length); // 5

Note: .length is a property, not a function.


arr.length   // ✅ correct
arr.length() // ❌ incorrect

Special use: You can modify .length to shrink or clear an array.


const arr = [1, 2, 3, 4, 5];
arr.length = 3;   // Shrinks array
console.log(arr); // [1, 2, 3]

arr.length = 0;   // Clears array
console.log(arr); // []

3. Mutability and const

Declaring an array with const prevents reassigning the variable, but contents can still change.


const arr = [1,2,3];
arr[0] = 99;      // ✅ works
arr = [4,5,6];    // ❌ error

4. Freezing Arrays: Object.freeze()

Freezes an array, preventing modifications. Freezing is shallow.


const arr = [1,2,3];
Object.freeze(arr);

arr[0] = 99; // ❌ ignored
arr.push(4); // ❌ TypeError
console.log(arr); // [1,2,3]
  • Works in-place, returns same reference
  • All references are frozen
  • Use deep freeze for nested objects

5. Tuples in JavaScript

JavaScript has no native tuple type. Use Object.freeze(array) to mimic.


const tuple = Object.freeze([10,20]);
tuple[0] = 99; // ❌ Won't change

6. Sets in JavaScript

Equivalent to Python sets. Store unique values.


const mySet = new Set([1,2,3,3]);
console.log(mySet); // Set(3) {1,2,3}
mySet.add(4);
console.log(mySet.has(2)); // true

7. Loops and Arrays

for...of → iterates over values.
for...in → iterates over keys/properties.


const arr = [10, 20, 30];

for (let val of arr) console.log(val); // 10,20,30
for (let key in arr) console.log(key); // 0,1,2
